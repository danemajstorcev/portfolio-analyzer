Portfolio Analyzer


Project Description:

A data visualization tool that analyzes and displays portfolio-related metrics in a clear and interactive way.


Tech Stack / Built With:

- React
- TypeScript
- CSS
- Charting Libraries


Features:

- Interactive data visualization
- Dynamic charts and metrics
- Clean analytical dashboard
- Responsive layout

 
Getting Started / Installation:

- git clone https://github.com/danemajstorcev/portfolio-analyzer.git
- cd portfolio-analyzer
- npm install
- npm run dev
